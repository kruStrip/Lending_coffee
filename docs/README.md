Это современный, быстрый и отзывчивый веб-сайт для кофейни «Арома Кафе». Проект включает в себя полноценный каталог товаров, интерактивную корзину, систему отзывов и функциональный блог.

## Основные возможности

### 🛒 Интернет-магазин (Каталог и Корзина)

-  **Динамический каталог**: Все товары (кофе, напитки, выпечка) управляются из JavaScript, что позволяет легко обновлять меню.
-  **Фильтрация**: Быстрый поиск по категориям с плавной анимацией.
-  **Интерактивная корзина**: Добавление товаров, изменение количества и удаление. Состояние корзины сохраняется в `localStorage`, поэтому покупки не пропадут при перезагрузке страницы.
-  **Оформление заказа**: Удобная форма с указанием номера столика.

### 📝 Блог и Комьюнити

-  **Индикатор чтения**: Визуальная полоска в верхней части страницы, показывающая прогресс прочтения статьи.
-  **Фильтрация статей**: Поиск контента по темам (Рецепты, История, Обжарка).
-  **Система комментариев**: Возможность оставлять отзывы под статьями, отвечать на сообщения других пользователей и ставить лайки.

### ⭐ Отзывы и Контакты

-  **Живые отзывы**: Пользователи могут оставлять свои впечатления прямо на главной странице.
-  **Обратная связь**: Работающая форма контактов для бронирования столиков или вопросов.

## Технологии

-  **HTML5**: Семантическая верстка.
-  **CSS3**: Использование переменных, Flexbox и Grid для адаптивности. Анимации для плавности интерфейса.
-  **JavaScript (Vanilla)**: Логика магазина, блога и модальных окон без тяжелых фреймворков.
-  **Font Awesome**: Современные иконки для удобной навигации.

## Как запустить проект

1. Склонируйте или скачайте файлы проекта.
2. Откройте файл `index.html` в любом современном браузере.
3. Для корректной работы путей изображений рекомендуется использовать локальный сервер (например, расширение Live Server в VS Code или `http-server`).
